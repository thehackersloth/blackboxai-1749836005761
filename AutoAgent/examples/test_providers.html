AutoAgent LLM Provider Integration

Ollama Integration

Configuration

OLLAMA_ENDPOINT=http://localhost:11434  # Optional

Available Models

  • phuzzy/darknemo:latest (recommended)
  • codellama:7b
  • llama2
  • Any other models pulled to your Ollama server

Usage Example

from autoagent import MetaChain
from autoagent.types import Agent

agent = Agent(
    name="ollama_agent",
    model="ollama/phuzzy/darknemo:latest",
    instructions="You are a helpful assistant."
)

chain = MetaChain()
response = chain.run(
    agent=agent,
    messages=[{"role": "user", "content": "Hello!"}]
)
